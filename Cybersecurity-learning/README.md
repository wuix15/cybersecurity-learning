
# **CyberSecurity learning — Notes & Writeups**

Parcours autodidacte de 0 à expert en cybersécurité — CTF, HackTheBox, RootMe, OverTheWire et plus.  
Ce repo est public et conçu pour que n'importe qui puisse apprendre à partir de mes notes.


## Objectif

Passer de débutant complet à expert en cybersécurité

## Ressources gratuites pour débuter

Si tu tombes sur ce repo et veux apprendre, voici le meilleur chemin :

### Plateformes d'entraînement

- [OverTheWire](https://overthewire.org/wargames/) — Linux & shell, commence par **Bandit**
- [TryHackMe](https://tryhackme.com) — Parcours guidés, idéal pour débuter
- [RootMe](https://www.root-me.org) — Catalogue de challenges classés par catégorie
- [HackTheBox](https://www.hackthebox.com) — Machines réalistes, niveau intermédiaire+
- [PicoCTF](https://picoctf.org) — CTF permanent, parfait pour les débutants

### Apprendre la théorie

- [PortSwigger Web Academy](https://portswigger.net/web-security) — Le meilleur cours web gratuit qui existe
- [CryptoHack](https://cryptohack.org) — Cryptographie par la pratique
- [pwn.college](https://pwn.college) — Binary exploitation de zéro
- [GTFOBins](https://gtfobins.github.io) — Référence pour la privesc Linux
- [HackTricks](https://book.hacktricks.xyz) — Bible des techniques offensives

## Avertissement légal

Tous les challenges documentés ici sont réalisés sur des **environnements légaux et autorisés** (plateformes CTF, labs dédiés). N'utilisez jamais ces techniques sur des systèmes sans autorisation explicite.
